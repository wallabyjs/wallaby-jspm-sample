# wallaby-jspm-sample
This repository demonstrates how to use [wallaby.js](http://wallabyjs.com/) with [jspm](http://jspm.io/) plus [Electron test runner](https://wallabyjs.com/docs/integration/electron.html).

## Install dependencies
```sh
npm install
jspm install
```
